# 🚗 Traffic Simulation & Car Parking Management System
## 📌 Overview
The **Traffic Simulation & Car Parking Management System** is a console-based application developed in **C++** as a university Data Structures project. It simulates vehicle movement at a traffic signal and manages parking operations using fundamental data structures.
The project demonstrates how different data structures can be applied to solve real-world traffic and parking management problems efficiently.

---
## ✨ Features
- Add cars to a traffic signal queue.
- Simulate green light operations.
- Park vehicles in the parking lot.
- Search vehicles using their license plate number.
- View all cars waiting at the traffic signal.
- View all parked cars.
- Prevent duplicate license plate numbers from being parked.
- Interactive menu-driven interface.
  
---
## 🛠 Data Structures Used
### Queue
Used to manage vehicles waiting at the traffic signal using the First-In-First-Out (FIFO) principle.

### Stack
Used to manage parked vehicles using the Last-In-First-Out (LIFO) principle.

### Binary Search Tree (BST)
Used to store and search vehicle records efficiently based on license plate numbers.

## ⚙️ How the System Works
1. A vehicle arrives and joins the traffic signal queue.
2. When the signal turns green, the vehicle moves forward.
3. The vehicle is parked in the parking lot.
4. The license plate is stored in the Binary Search Tree for fast searching.
5. Users can search for any parked vehicle using its plate number.
6. Vehicles leave the parking lot in LIFO order.
